## [1.0.0] - 2026-06-01

### Added

**Frontend:**
- Angular 18 + Tailwind CSS SPA
- Menu display with category filtering
- Chef profiles showcase
- Reservation booking system
- Reviews & ratings
- Contact form with validation
- Animated UI transitions

**Backend:**
- ASP.NET Core 8 REST API
- Clean Architecture (CQRS + MediatR)
- Entity Framework Core + SQL Server
- Swagger/OpenAPI docs
- FluentValidation input validation
- Global exception handling
- Health check endpoint

**Infrastructure:**
- Docker + Docker Compose support
- Vercel (frontend) and Azure/Docker (backend) deployment
- CI/CD pipeline configuration

---

## Version Policy

- **MAJOR** — breaking API changes  
- **MINOR** — backwards-compatible new features  
- **PATCH** — bug fixes  

Support: Current `1.x` receives full updates. Older majors receive security fixes for 6 months only.
